# Media Corpus Licensing

Every file under `sources/` is **generated by this project** (see the
generation recipe in the PR that introduced it) and dedicated to the public
domain under CC0-1.0:

| File | Generation |
|---|---|
| `speech-tone.wav` | Python `wave` module — 1.5s 16kHz mono dual-tone sine |
| `chart.png` | Hand-built minimal PNG (zlib/struct) — 64×64 RGB gradient |
| `clip.mp4` | ffmpeg `testsrc` + `sine` lavfi filters, 2s 160×120 |
| `two-page.pdf` | Hand-built minimal PDF, two pages of Helvetica text |
| `post.html` | Hand-written synthetic OpenGraph post |

No third-party media is included. Real CC0 speech (for WER gates) arrives
with the audio modality phase and must be recorded here with its provenance.
